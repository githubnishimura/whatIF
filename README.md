
■ ゲーム概要

本ゲームは、場のカードと条件が合う手札を出し続け、できるだけ多くのカードを捨てることを目的とする一人用カードゲームです。


■ カードを出せる条件

次のいずれかを満たす場合、カードを出せます。

1. 場のカードと同じスート
2. 場のカードと同じ数字

例：
場が「♠7」の場合
・♠なら何でも出せる
・7なら他スートでも出せる


■ ターンの流れ

1. 出せるカードを選ぶ
2. そのカードを場に出す
3. 山札から1枚補充する
4. 出せるカードが無くなるまで続ける

出せるカードが無くなった時点で終了です。


■ カード状況表示の見方

画面に表示されるカード一覧は、以下の記号で表されます。

■ ＝ まだ捨てられていないカード
□ ＝ すでに捨てられたカード
H ＝ 手元のカード

これにより、ゲーム全体の進行状況を視覚的に確認できます。


■ 操作方法

マウスか画面タッチでカードを選びます。
カーソルがカード上にあると、場にだせるカードはすこし動きます。

■ 攻略のコツとは

本ゲームにおける「攻略のコツ」とは、複数の出せるカードの中から、
継続してカードを出し続けられる可能性が最も高い選択を行うことです。

■ 期待値の考え方

「期待値」とは、その選択をした場合に今後どれだけ続けてカードを出せる可能性があるかという見込みを意味します。

考えるポイント：

・手札の中で同じスートが多いか
・同じ数字が複数あるか
・すでに多く捨てられている数字かどうか

一手先だけでなく、「その次につながるか」を意識することが重要です。
